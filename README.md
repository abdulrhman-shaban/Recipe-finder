# Recipe Finder App 🧑‍🍳

A clean, interactive web application that allows users to search for recipes from all over the world. It provides detailed instructions, ingredients, and video tutorials for a great cooking experience.

## Description
**Recipe Finder** is a web application designed to help food enthusiasts discover delicious recipes easily. Powered by **TheMealDB API**, this app provides a seamless search experience, allowing users to explore a vast collection of meals based on their preferences.

## Key Features
- **Search Functionality:** Instantly search for recipes by name or main ingredient.
- **Detailed View:** Access full recipe details including preparation instructions and ingredient lists.
- **Video Integration:** Watch cooking demonstrations directly via integrated YouTube links.
- **Responsive Design:** A mobile-friendly interface built with CSS Grid and Flexbox for an optimal experience on any device.

## Technologies Used
- **HTML5:** Semantic structure of the application.
- **CSS3:** Modern styling using Flexbox, CSS Grid, and custom variables.
- **JavaScript (ES6+):** Asynchronous programming, Fetch API, and DOM manipulation.
- **TheMealDB API:** Used to fetch real-time recipe data.
- **Font Awesome:** For iconography.
